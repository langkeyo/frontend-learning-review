## 概念

HTML5 引入了语义化标签，让页面结构更加清晰。语义化标签让浏览器、搜索引擎、屏幕阅读器更好地理解页面内容。

### 核心语义化标签

| 标签 | 用途 | 语义 |
|------|------|------|
| `<header>` | 页面或区块头部 | 导航、标题、Logo |
| `<nav>` | 导航菜单 | 链接集合、菜单 |
| `<main>` | 主要内容区 | 页面核心内容 |
| `<article>` | 独立文章 | 可独立分发的内容 |
| `<section>` | 内容区块 | 主题相关的内容组 |
| `<aside>` | 侧边栏 | 辅助信息、广告 |
| `<footer>` | 页面或区块底部 | 版权、联系方式 |

### 为什么需要语义化标签？

1. **SEO 优化**：搜索引擎能更好地理解页面内容结构
2. **可访问性**：屏幕阅读器用户能正确导航和理解页面
3. **代码可读性**：开发者更容易理解页面结构
4. **样式分离**：减少对 `class` 的过度依赖
